# Product Images

This repository contains product images for an e-commerce dataset.

## Structure

Images are organized by category:

```
├── tops/
├── dresses/
├── shirts/
├── jeans/
├── sports-shoes/
└── ... (86 categories total)
```

## Usage

Images are hosted via GitHub Pages and can be accessed at:

```
https://pazhanir.github.io/product-images/{category}/{product_id}_{image_number}.jpg
```
